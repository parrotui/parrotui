# ParrotUI 更新日志

## [0.1.0] - 2026-03-07

### 新增

#### 基础组件
- `ParrotIcon` - 图标组件
- `ParrotButton` - 按钮组件
- `ParrotPrice` - 价格展示组件

#### 导航组件
- `ParrotNavBar` - 导航栏组件
- `ParrotTab` - 标签切换组件
- `ParrotFooter` - 页脚组件
- `ParrotFooterLinks` - 页脚链接组件

#### 表单组件
- `ParrotSearchBar` - 搜索栏组件（支持分类选择、相机按钮）
- `ParrotSelector` - 选择器组件
- `ParrotFilterDrawer` - 筛选抽屉组件

#### 展示组件
- `ParrotNoticeBar` - 公告栏组件（支持轮播）
- `ParrotIndexCard` - 指数卡片组件
- `ParrotProductCard` - 产品卡片组件
- `ParrotTradeItem` - 交易项组件
- `ParrotTimeline` - 时间轴组件

#### 业务组件
- `ParrotTmClassSelector` - 商标分类选择器
- `ParrotTmStatusTag` - 商标状态标签
- `ParrotApplicantCard` - 申请人卡片

### 规范

- 统一使用 `.uvue` 文件格式
- 统一使用 `<script setup lang="uts">` 语法
- 统一使用扁平 class 选择器
- 统一组件命名规范（ParrotKebabCase）
- 统一目录命名规范（parrot-kebab-case）

### 文档

- 添加 README.md 总说明文档
- 添加 PACKAGE.md 打包文档
- 添加 LICENSE MIT 开源协议
- 添加 .gitignore 文件

- 每个组件配备独立的 README.md 使用文档
