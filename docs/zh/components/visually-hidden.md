# VisuallyHidden 视觉隐藏组件

视觉隐藏组件，无障碍隐藏元素（屏幕阅读器可见）。

## 基础用法

```vue
<template>
  <parrot-visually-hidden />
</template>
```

## API

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
无 Props，仅包裹需要视觉隐藏的内容。

### Slots

| 插槽 | 说明 |
|------|------|
| `default` | 默认内容 |
