# Clipboard 剪贴板组件

剪贴板组件，剪贴板读写操作。

## 基础用法

```vue
<template>
  <parrot-clipboard />
</template>
```

## API

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `text` | `string` | `''` | 要复制的文本 |

### Slots

| 插槽 | 说明 |
|------|------|
| `default` | 默认内容 |
