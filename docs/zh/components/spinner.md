# Spinner 加载旋转组件

加载旋转组件，多种旋转样式。

## 基础用法

```vue
<template>
  <parrot-spinner />
</template>
```

## API

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `size` | `string` | `'48rpx'` | 尺寸 |
| `color` | `string` | `'#1a7eff'` | 颜色 |

### Slots

| 插槽 | 说明 |
|------|------|
| `default` | 默认内容 |
