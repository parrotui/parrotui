# Introduction

**Parrot UI** is a cross-platform component library built on uni-app X, supporting Android, iOS, HarmonyOS, WeChat MiniProgram, and Web from a single codebase.

## Features

- **Vue3 + Composition API** — Modern development experience with UTS strict type checking
- **Design Token System** — Complete tokens for colors, spacing, typography, and shadows
- **Dynamic Themes** — Admin panel can update brand colors globally at runtime
- **Dark Mode** — Built-in light/dark switching with smooth transitions
- **Cross-Platform** — Android / iOS / HarmonyOS / WeChat MiniProgram / Web
