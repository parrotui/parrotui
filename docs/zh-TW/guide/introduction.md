# 介紹

**Parrot UI** 是基於 uni-app X 的跨終端元件庫，一套程式碼支援 Android、iOS、鴻蒙、微信小程式和 Web。

## 特性

- **Vue3 + Composition API** — 現代化開發體驗，UTS 嚴格型別檢查
- **設計令牌體系** — 完整的顏色、間距、字體、陰影令牌
- **動態主題** — 後台可在執行時動態修改品牌色，全域即時生效
- **暗黑模式** — 內建亮色/暗色切換，平滑過渡
- **跨終端** — Android / iOS / 鴻蒙 / 微信小程式 / Web
